KissMT Dynamic SEO Meta Tags version 1.0

Installation

Remove any previous meta tags contribution

1 - Uploads

Extract zip file and copy resulting folder to catalog/includes/modules/

Ensure that the following directory is writeable (0755 or 0777 dependent on server) catalog/includes/modules/kiss_meta_tags/cache/

2 - File modifications

Perform the following three steps to your current template, the default is catalog/templates/default.php

- Find ..

<title><?php echo STORE_NAME . ($osC_Template->hasPageTitle() ? ': ' . $osC_Template->getPageTitle() : ''); ?></title>

- Replace with ...

<?php
// Begin Dynamic Meta Tag Code
if (file_exists('includes/modules/kiss_meta_tags/kiss_meta_tags.php'))
{
  include_once('includes/modules/kiss_meta_tags/kiss_meta_tags.php');
}
// End Dynamic Meta Tag Code
?>

- Find and remove ..

if ($osC_Template->hasPageTags()) {
  echo $osC_Template->getPageTags();
}

- Find and remove ..

<meta name="Generator" content="osCommerce" />

3 - Static Tags

Make changes to catalog/includes/modules/kiss_meta_tags/languages/(code)/kissmt.php to suit your site.

You can add other .php files names to the Static Tag "Generic Pages" using the same format if you wish.

Once you have completed the above KissMT is already producing complex SEO meta and canonical tags on your site ... Go and check .. view some categories/products and choose a manufacturer from the dropdown. View source to see the keywords and description.

Important Information

KissMT has code in place to ensure that www.mysite.com/ and www.mysite.com/index.php is not duplicate content.
The canonical version as standard is www.mysite.com/ ( recommended ) if you have .htaccess redirects to achieve this they can now be removed.
If however you are using or would like to use www.mysite.com/index.php as the root canonical link you will need to adjust abstracts/kiss_modules.php protected $root_index ( near the top )
